How to create a webpage

  • Have an index.html file with a css file linked to it for text animations.
  • Create an sftp account
  • Go into terminal and start up sftp with the code sftp username@cmis-hsteacher
  • Upload your index and css files from your terminal into the sftp with command put.
  • When you upload your sftp, check with command ls to see if the correct file has been uploaded.
  • After uploading to sftp, git add commit and push the files to your repository to save them.
  • To view the file on the local computer, use local host, however, to view the files on the production server, use cmis-hsteacher/~username/foldername.
